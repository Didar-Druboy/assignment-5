Important FAQ

What is the Document Object Model (DOM)?

DOM is a programming interface. It represents the structure of a document as like a tree of objects, where we can get node and element in the HTML document. Through the DOM we can interact with and manipulate the HTML content, structure, and style.

How do you select an element from the DOM?

We can select an element by using

  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()
  • querySelector() and
  • querySelectorAll()
    • What is event delegation in the context of the DOM, and why is it useful?

    Event delegation is a technique in JavaScript where we can take advantage of event bubbling to handle events efficiently. It helps us to catch the childrens of an element by using a single event listener to its parent instead of attaching event listeners to multiple child elements and then use that listener to handle events from its children. This is particularly useful when dealing with dynamically added elements.

    How do you manipulate an element's attributes and styles using the DOM?

    There are three ways to manipulate attributes as like

  • getAttribute()
  • setAttribute()
  • removeAttribute()

    • We can use .style .CSS-Property-Name and give it a value as like document.getElementById('idName').style.backgroundColor = 'red'

    Also we can add or remove a class name by using

  • document.getElementById('idName').classList.add('className')
  • document.getElementById('idName').classList.remove('className')
